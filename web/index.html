Grafana Integration

This API is fully compatible with the SimpleJSON or Infinity data source plugins in Grafana.

  1. Install JSON API or Infinity datasource in Grafana.
  2. Add new Datasource and set the URL to:
    http://your-server-ip:9191
  3. Define query URL path as: /api/v1/xport
  4. Configure headers or parameters if Basic Auth or HMAC signing is enabled.
# API Signature Authentication Example (Go) # Query parameters are hashed using HMAC-SHA256 key := []byte("your_secret") message := "/api/v1/xport?start=-1h&x=1751912400&" mac := hmac.New(sha256.New, key) mac.Write([]byte(message)) signature := hex.EncodeToString(mac.Sum(nil))
